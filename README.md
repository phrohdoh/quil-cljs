# clojurescript quil (via shadow-cljs)

## running

in terminal 1:
```
npx shadow-cljs server
```

in terminal 2:
```
npx shadow-cljs watch app
```

visit http://localhost:8020

### requirements

- clj tools.deps 
- npm/yarn
- shadow-cljs (install js deps via npm/yarn)
- 2 terminal instances
